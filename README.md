# 📅 Планировщик студента v4 — "Midnight Glass"

## 🚀 Запуск в PyCharm

1. Открой папку `student_planner_v3` как проект
2. Интерпретатор: Python 3.10+
3. Запусти **`main.py`**
4. Войди с тестовым аккаунтом или зарегистрируйся

> Никаких pip install — только стандартная библиотека Python.

## 📧 Настройка email-уведомлений

Открой `services/email_service.py`, найди `EMAIL_CONFIG` и вставь свой Gmail:

```python
EMAIL_CONFIG = {
    "sender_email":    "yourname@gmail.com",
    "sender_password": "xxxx xxxx xxxx xxxx",  # App Password
    "enabled": True,
}
```

Получить App Password: Gmail → Настройки → Безопасность → Пароли приложений

## ✅ Что умеет приложение

### Студент
- Личный кабинет с sidebar-навигацией
- Создание и управление задачами (ДЗ / лаб / экзамен)
- Сдача работ с прикреплением файлов + комментарий
- Просмотр оценок и отзывов преподавателя
- Профиль со статистикой

### Преподаватель
- Все задания с кол-вом сдач
- Назначение задания конкретному студенту
- Страница «На проверку» — все сдачи студентов
- Просмотр файлов и оценивание с отзывом
- Список студентов со статистикой
- Дашборд с графиками по предметам

### Общее
- 🔐 Регистрация / авторизация
- 📧 Email-уведомления (регистрация, назначение, сдача, оценка)
- 🔔 Push-уведомления (Windows / macOS / Linux)
- 📎 Загрузка файлов к задачам и сдачам
- 🔍 Поиск и фильтрация задач
- 🎨 Современный тёмный UI "Midnight Glass"
